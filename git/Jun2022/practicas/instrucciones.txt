# Instrucciones tarea final:

	1. bifurcar el repositorio git@github.com:josiasvasquez/itgrarte.git
	2. clonar el repositorio bifurcado a nuestro equipo local,
	3 crear una rama con nuestro nombre,
	4. cambiarnos a la rama creada.
	5. Crear un archivo con nuestro nombre.
	6. en el archivo poner la siguiente información según nuestra elección.
		A) Que opinión tenemos sobre el curso de git y github,
		B). Que sugerimos para mejorar el curso,
		c). Como aplicaremos los conocimientos aprendidos en este curso.
		d). Enlace a nuestro usuario de github,
	7. Guardar los cambios,
8. Agregar el archivo con git add .
9. hacer un commit,
10 enviar los cambios a el repositorio bifurcado en github.
11. Enviar una pull request solicitando se agregue nuestra información al repositorio.

Nota: De preferencia usar archivos html o md, si no, usar archivos txt.
